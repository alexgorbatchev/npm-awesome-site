videoconverter.js

posted by Alex Gorbatchev on November 22nd 2013

videoconverter.js is not really an NPM module, but is a library that allows you to convert and manipulate videos inside of your web browser. This is achieved by converting the popular FFmpeg library into JavaScript, using Emscripten. In other words, it’s damn impressive! Just check out the demo.

Here’s the fun fact, ffmpeg.js is a whoping 29MB in size. GZipped it comes down to 6MB. I have tried using UglifyJS to minify the source and gave up after 1 hour.

There are plenty of examples and information on the github page.
