node-dependencies

posted by Alex Gorbatchev on February 10th 2014

node-dependencies is a very basic but incredibly useful command line utility written by Doug Patti of Fog Creek that checks and prints out out-of-date dependencies for your Node.js app.

npm install node-dependencies

Usage
